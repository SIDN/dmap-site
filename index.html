Source:[ Wikipedia](https://en.wikipedia.org/wiki/File:Orbis_Terrarum_Nova_et_Accuratissima_Tabula_by_Nicolaes_Visscher,_1658.jpg)](img/map2.jpg) Dmap is multi-application crawler that *reduces the complexity* in both executing Internet measurements and analyzing their results. It takes as input a list domain names (entire DNS zones even), and: 1. *Crawls* domains for DNS, HTTP, HTTPS, TLS, SMTP and a generates a screenshot of a webpage (if it's available) 1. *Converts* the raw measurement data and store into a 166-features unified SQL data model, so users can easily and convienlty analyze the results using SQL. We have been using Dmap since December 2017 in the analysis of the .nl DNS zone. Just run it and query it. More details on our [research paper](paper.pdf). **NEWS:** * **2018-06-05**: website online # How to get it We have developed [Dmap](http://dmap.sidnlabs.nl) as a project with [SIDN Labs](https://sidnlabs.nl), the research arm of [SIDN](https://sidn.nl), the .nl top-level domain registry. We are more than happy to make Dmap availble for researchers, both code and binaries. We just would not like it to be used commercially. Therefore, if you are a researcher, just do the following: 1. Send a request to `sidnlabs at sidn dot nl` with the following information: * Full Name * University/Research Center/Research Group * Your e-mail address (from your University/Research Center) * No Gmail, Yahoo or similar please. * Github account name (to give you access to the repository) 1. We will send you our academic license agreement (Modified Apache 2 License) 1. Once you send it back to us, we will quickly process that and give you full access to our [GitHub Repository](https://github.com/SIDN/emap). # [Demo data page](demo.html) We have a database dump from Alexa 1M domains, which we cover our [research paper](paper.pdf). Please check our [demo data page](demo.html), which provides both data and data model. ## [Data Model (from raw data to SQL)](datamodel.html) Very often, researchers spend an awful lot of time planning and executing measurements. The complexity of different data formats and issues that may emerge drain energy that could be otherwise be spent on research questions. Dmap is intended to be an enabling platform that free researchers from the complexity of measurements and parsing complex data formats; rather, it automates the measurements and provides a SQL interface o the data. Want to give at try? Check our [data demo](demo.html) page and the [our data model page](datamodel.html) data shows how the results are stored in the SQL database. # [Research paper](paper.pdf) For a full description, please [download our research paper (PDF).](paper.pdf) ```verbatim Wullink, M., Moura, G.C.
